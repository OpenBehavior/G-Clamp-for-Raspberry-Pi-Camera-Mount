# G-Clamp-for-Raspberry-Pi-Camera-Mount

Clamp for a Raspberry Pi cam mount.

Designed by Elizabeth Glover (ejglover@uic.edu)

Universal G-Clamp for mounting Raspberry Pi cameras and IR LED units on a variety of structures, such as ring stands, or wire shelves.

Original posting: https://www.thingiverse.com/thing:5383893

This design was validated by Kevin Chávez-López on 05/23 on behalf of the OpenBehavior project.
